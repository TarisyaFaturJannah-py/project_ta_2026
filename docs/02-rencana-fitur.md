Rencana Fitur

Fitur 1 Dashboard E-Book

Role Penanggung Jawab: Backend, Frontend

Sumber Data: Internal System

Deskripsi & Ekspektasi:
Pengguna dapat melihat halaman utama yang menampilkan daftar buku terbaru, buku populer, dan kategori buku. Dashboard menjadi halaman awal yang memudahkan pengguna menjelajahi koleksi e-book.

Fitur 2 Pencarian & Filter Buku

Role Penanggung Jawab: Frontend, Backend

Sumber Data: Internal System

Deskripsi & Ekspektasi:
Pengguna dapat mencari buku berdasarkan judul atau penulis serta melakukan filter berdasarkan kategori.

Fitur 3 Detail & Preview Buku

Role Penanggung Jawab: Frontend, Backend

Sumber Data: Internal System

Deskripsi & Ekspektasi:
Pengguna dapat melihat informasi lengkap buku seperti cover, penulis, deskripsi, kategori, serta melakukan preview atau download e-book.

Fitur 4 Dashboard Admin

Role Penanggung Jawab: Frontend, Backend

Sumber Data: Internal System

Deskripsi & Ekspektasi:
Admin dapat mengelola seluruh data buku melalui dashboard khusus. Dashboard menampilkan statistik serta menu CRUD buku dan kategori.

Fitur 5 Integrasi Metadata Buku

Role Penanggung Jawab: Frontend, Backend

Sumber Data: Third-Party API (Open Library)

Deskripsi & Ekspektasi:
Saat admin memasukkan ISBN atau judul buku, sistem mengambil metadata otomatis dari Open Library seperti cover, penulis, dan deskripsi buku.
